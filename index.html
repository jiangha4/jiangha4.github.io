    Cerberus

    Secrets Manager Microservice

    Cerberus is an open source cloud first microservice used as a secrets manager for deploy and run time secrets management. Cerberus can be used to store both key value pair secrets and files such as pem files. Cerberus is built using a Java Springboot framework and deployed to AWS. Along with a UI, Cerberus also comes with a multiple clients to programmatically interface with Cerberus.

    Cerberus is deployed in multiple regions within AWS to allow for a hot swap in case of a main region failover event.

    Referee

    UI for Canary Deployments

    Referee is an open source UI for automated canary deployments using Spinnaker's Kayenta.

    Canary deployments is a deployment pattern where a small subset of user traffic is directed to the new deployment stack and it's state is tested against the current existing stack. If the new state is the same as the original stack, then the new stack is promoted and all traffic is directed to it and the original stack is taken down.
